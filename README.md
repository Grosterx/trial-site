# FlexFlow - Digital агентство и Design студия

Веб-сайт для digital агентства FlexFlow, специализирующегося на создании эффективных цифровых решений для бизнеса.

## Описание

Одностраничный веб-сайт, представляющий услуги digital агентства FlexFlow. Сайт включает в себя информацию об услугах компании, портфолио проектов, информацию о компании и контактные данные.

## Технологии

- **HTML5** - структура сайта
- **CSS3** - стилизация (файл `style.css`)
- **Bootstrap 5.3.3** - адаптивная верстка и компоненты
- **Google Fonts** - шрифт Roboto
- **JavaScript** - Bootstrap JS для интерактивных элементов

## Структура проекта

```
digital агентство/
├── index.html          # Главная страница
├── style.css           # Пользовательские стили
├── images/             # Изображения и иконки
└── README.md          # Документация проекта
```

## Основные разделы сайта

1. **Шапка (Header)** - навигационное меню с логотипом и ссылками
2. **Главный блок (Hero)** - приветственный блок с основным заголовком
3. **Секция компаний** - логотипы партнеров
4. **Услуги** - описание предоставляемых услуг:
   - Веб и мобильные приложения
   - Индивидуальный дизайн
   - Искусственный интеллект
5. **Индивидуальные решения** - информация о персонализированных решениях
6. **Профессиональная экспертиза** - описание экспертизы компании
7. **Портфолио** - примеры выполненных проектов
8. **Начало работы** - призыв к действию
9. **Подвал (Footer)** - контакты и социальные сети

## Особенности

- ✅ Адаптивный дизайн (responsive)
- ✅ Мобильное меню с бургер-кнопкой
- ✅ Современный и чистый интерфейс
- ✅ Оптимизированная структура кода

## Запуск проекта

1. Откройте файл `index.html` в браузере
2. Или используйте локальный сервер (например, Live Server в VS Code)
